GPT-Request
=================
OPEN IA request API.

---
## Overview  
This API is responsible for communicating with Openai and returning responses asynchronously.

technologies used:

- Java 17
- Spring Boot 3.3.0
- Webflux (reactor)
---

## Main class:

``` java
com.spring.gpt.GptApplication
```

## Installation

---
```
mvn clean
mvn install
```
